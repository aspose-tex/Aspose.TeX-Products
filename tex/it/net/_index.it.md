<p>
TeX è un linguaggio di programmazione, potresti trovarlo chiamato data markup one. Allo stesso tempo, TeX è un motore di interpretariato che comprende questo linguaggio e un sistema di composizione nel suo insieme.
Viene utilizzato per la scrittura di testi matematici o tecnici. Molti pacchetti differenti sono stati creati sulla base di ΤeΧ. Il più famoso di questi pacchetti è LaTeX. Consente agli autori di concentrarsi sul testo e di lasciare il processo tipografico al programma. Con LaTeX i documenti strutturati sono facili da creare e gestire.
</p>
<p>
Aspose.TeX per .NET è una libreria flessibile e facile da usare per comporre file TeX e LaTeX. L'API supporta più formati di output come XPS, PDF, PNG, JPEG, TIFF, SVG, BMP. Significava che con la funzionalità di questa libreria puoi convertire file TeX e LaTeX in qualsiasi formato di output. Con la soluzione, puoi creare un formato TeX personalizzato se desideri creare e/o comporre una serie di documenti progettati in modo uniforme.
</p>
