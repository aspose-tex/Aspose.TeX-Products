<p>
TeX adalah bahasa pemrograman, Anda mungkin menemukannya sebagai markup data. Pada saat yang sama, TeX adalah mesin penerjemah yang memahami bahasa ini dan sistem pengaturan huruf secara keseluruhan.
Ini digunakan untuk penulisan teks matematika atau teknis. Banyak paket berbeda telah dibuat berdasarkan eÎ§. Paket yang paling terkenal adalah LaTeX. Ini memungkinkan penulis fokus pada teks dan menyerahkan proses tipografi ke program. Dengan LaTeX, dokumen terstruktur mudah dibuat dan dikelola.
</p>
<p>
Aspose.TeX untuk C++ adalah pustaka yang fleksibel dan mudah digunakan untuk mengeset file TeX dan LaTeX. API mendukung beberapa format output seperti XPS, PDF, PNG, JPEG, TIFF, SVG, BMP. Ini berarti bahwa dengan fungsionalitas perpustakaan ini Anda dapat mengonversi file TeX dan LaTeX ke salah satu format output. Dengan solusinya, Anda dapat membuat format TeX kustom jika Anda ingin membuat dan/atau mengeset banyak dokumen yang dirancang secara seragam. 
</p>
