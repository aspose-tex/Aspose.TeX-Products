<p>
TeX é uma linguagem de programação, você pode encontrá-la chamada de marcação de dados. Ao mesmo tempo, o TeX é um mecanismo de interpretação que entende essa linguagem e um sistema de composição como um todo.
É usado para escrever textos matemáticos ou técnicos. Muitos pacotes diferentes foram criados com base em ΤeΧ. O pacote mais famoso é o LaTeX. Ele permite que os autores se concentrem no texto e deixem o processo de tipografia para o programa. Com o LaTeX, os documentos estruturados são fáceis de criar e gerenciar.
</p>
<p>
Aspose.TeX for .NET é uma biblioteca flexível e fácil de usar para formatar arquivos TeX e LaTeX. A API suporta vários formatos de saída como XPS, PDF, PNG, JPEG, TIFF, SVG, BMP. Isso significa que com a funcionalidade desta biblioteca você pode converter arquivos TeX e LaTeX para qualquer um dos formatos de saída. Com a solução, você pode criar um formato TeX personalizado se quiser criar e/ou compor vários documentos projetados de maneira uniforme.
</p>
